# --- CALCULADORA ANDROID ---

Se trata de una simple calculadora pensada para su uso en Android. 

## La apliación ofrece:

    1. Un par de campos a rellenar (Nº1 y Nº2).
    2. Opciones básicas:
        3. Btn Suma
        4. Btn Resta
        5. Btn Multiplicación
        6. Btn División
    7. Btn de resteo de campos.
